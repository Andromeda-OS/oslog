# oslog
os_log command line tool implementation for iOS and PureDarwin

A tool that shows os_log stream info/activity directly on iOS devices and PureDarwin 

usage:

	oslog [--info|--debug] [ -p pid ] [--noLevelInfo] [--noSubsystemInfo]

	Examples:
		oslog 
		oslog --debug
		oslog -p SpringBoard
